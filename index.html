¡Bienvenido a BIZUNI APP!

Utilidades de la aplicación

  • Compraventa de productos entre universitarios.
  • Búsqueda de artículos específicos por categoría o palabras clave.
  • Chat integrado para comunicarte con los vendedores y realizar acuerdos.
  • Calificaciones y comentarios para mantener la confianza en las transacciones.
  • Notificaciones en tiempo real para estar al tanto de nuevas oportunidades.

Cómo empezar

Para comenzar a utilizar BIZUNI APP, sigue estos pasos:

  1. Descarga la aplicación desde nuestra página oficial.
  2. Crea una cuenta utilizando tu correo electrónico y contraseña.
  3. Explora los artículos disponibles o publica tus propios productos.
  4. Contacta a otros usuarios a través del chat para cerrar la transacción.
